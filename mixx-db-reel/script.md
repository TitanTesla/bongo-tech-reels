# Database — VO Script (final)

* Ukiweka 3,000 Tigo Pesa afu ukaenda kuangalia balance yako kwenye app. Jua kwamba hakuna sehemu kwenye ofisi za Tigo ambapo kuna kisanduku chenye pesa zako in cash.
* Najua hio ni common sense but what if nkikwambia kwamba unapomtumia mtu pesa hakuna actually unachotuma? Kinachofanyika ni kwamba database ina update number kwenye account mbili tofauti.
* Kama hapo awali ulikuwa na 3000TZS na Fatuma 2000TZS. Ukaamua kutuma 2000TZS kwa Fatuma, Database itapunguza namba kutoka kwenye table yako na kuongeza namba kwenye table ya Fatuma. ila Hakuna pesa iloingia wala kutoka ni namba tu zimebadilishwa.
* Database ni miundo mbinu inayohusika kuhifadhi taarifa zako. So kwako wewe mtamazamaji Namba yako ya simu, NIDA, chats zako, picha zako ziko kwenye database somewhere.
* Inaeza iwe nje ya nchi kwenye database za Whatsapp au instagram au ndani kwenye database za serikali.

Siku wakiripua hizi database kazi tunayo😂.

* Moja kati ya swali ambalo naulizwaga na wateja wangu wa apps especially watu wazima "Taarifa za mauzo au kodi ya nyumba zinahifadhiwa wapi mjukuu wangu?".
* Jibu ni database, ambayo inaeza kuwa document database kama MongoDB, relational database kama PostgreSQL au vector database kama chroma DB inayosaidia ML models au Ai kuelewa context ya unachokiuliza.
